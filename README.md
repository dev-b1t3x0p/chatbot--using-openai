# Chatbot Using OpenAI

## Overview
This project demonstrates how to create a chatbot using OpenAI's API. The chatbot can answer questions, engage in conversations, and provide information based on user input.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dev-b1t3x0p/chatbot--using-openai.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chatbot--using-openai
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
1. Make sure you have your OpenAI API key ready.
2. Set up your environment variables according to the `.env.example` file provided.
3. Start the chatbot:
   ```bash
   node index.js
   ```
4. Follow the on-screen instructions to interact with the chatbot.

## Features
- Engages in context-aware conversations
- Provides informative responses
- Easy to integrate with other applications
